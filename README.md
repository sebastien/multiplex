# Installing

Quick, from the shell:

```
$ curl -o multiplex https://raw.githubusercontent.com/sebastien/multiplex/main/src/py/multiplex.py; chmod +x multiplex
$ ./multiplex --help
```

alternatively, using `pip`:

```
$ python -m pip install --user multiple
```
